Random Thingz

Probably just another random things mod.

 

Starting with The World Items mod and it's versions 1, 2 and 3.

But now knowing more about creating mods, there's a new new version of the mod with a brand new name.

The mod has been rewritten in pure Java. And adding already more things that before: The QForgeMod Random Thingz Project!

 

Features

Currently 2 trees: cherry and eucalyptus including leaves, logs, planks and saplings

Two animals from announced in MineCon Live 2020: Moobloom and Glow Squid

Two special hostile mobs: Fire Creeper, Ice Enderman.

Some missing baby variants: creeper, stray, skeleton, wither skeleton, Enderman.

Ore generation including: ruby, topaz, nickel, uranium, copper, aluminum and more.

Two biomes, currently 1 that's a test biome.

Advanced debug menu, with a lot more options than the default one, default key-bind is F4.

A bunch of machines including quarry, electric furnace, coal generator, refinery, and more.

Special items including: fire sword, teleport wand, magnet and a battery.

God items: unstable infinity ingot, infinity sword, kill switch, walking staff.

There are since 1.1 Beta 11 22 different Moobloom variants.

Since 1.1 Beta 10 there are 225 different bookshelfs.

Bugs / Issues / Crashes

In Alpha / Beta releases stuff might change or break, it's recommended to use them for testing purposes only.

If there's a problem with the mod: don't post the crash report in the comments, but post it on the issue tracker here: Issues · Ultreon/RandomThingz (github.com).

If there's no crash report but it still crashes, post the debug.log file in your .minecraft/logs folder to the issue tracker.

If there's no crash report and there's no crash in the debug.log, then there's probably a problem with the Java installation or a windows driver (what I have sometimes).
